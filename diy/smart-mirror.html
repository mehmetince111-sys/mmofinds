Smart Mirror mit Raspberry Pi bauen

Baue deinen eigenen Smart Mirror — ein Spiegel, der Wetter, Kalender, Nachrichten und KI-Assistent-Funktionen zeigt. Ein echter Eye-Catcher für jedes Badezimmer oder Schlafzimmer.

💡 Was ein Smart Mirror kann

Dein Smart Mirror zeigt:

  • Wetter: Aktuelle Temperatur, Vorhersage, UV-Index
  • Kalender: Nächste Termine, Geburtstage
  • Nachrichten: KI-zusammengefasste Top-Nachrichten
  • Uhr: Große digitale Uhr mit Datum
  • Stimmung: KI-generierte Motivation/Zitat des Tages
  • Gesundheit: Schlaf-Tracking, Schritte, Herzfrequenz

📦 Materialliste

  • 🛒 Raspberry Pi 4 (4GB) — ~60€
  • 🛒 22" LCD Display — ~120€
  • Zwei-Weg-Spiegel-Folie — ~30€
  • LED-Streifen — ~15€
  • 🛒 Holzrahmen oder Acryl-Gehäuse — ~25€
  • USB-C Netzteil — ~10€

Gesamtkosten: ~260€

Schritt 1: Hardware zusammenbauen

Lege das LCD Display in den Rahmen, klebe die Zwei-Weg-Spiegel-Folie auf das Display und verbinde den Pi mit dem Display über HDMI. Die LED-Streifen werden hinter dem Spiegel platziert für Hintergrund-Beleuchtung.

Schritt 2: MagicMirror Software installieren

Installiere MagicMirror² — die leading Open-Source Smart Mirror Platform. Sie unterstützt über 200 Module für Wetter, Kalender, News, Musik, Fitness und mehr.

Schritt 3: Module konfigurieren

Konfiguriere die Module, die du verwenden willst. Die beliebtesten Module:

  • Weather — OpenWeatherMap API
  • Calendar — Google Calendar Integration
  • News — KI-zusammengefasste Nachrichten
  • Quote — Zufällige Motivation/Zitate
  • Health — Fitbit/Apple Health Integration

Schritt 4: KI-Integration

Integriere einen lokalen KI-Assistenten für sprachgesteuerte Queries. Du kannst Fragen wie "Wie wird das Wetter morgen?" oder "Was habe ich heute im Kalender?" stellen und der Mirror antwortet direkt.

✅ Was du bekommst

  • Ein echter Eye-Catcher für dein Zuhause
  • Alle wichtigen Infos auf einen Blick
  • Sprachgesteuerte KI-Queries
  • Keine Cloud-Abonnement nötig
  • Vollständig anpassbar

🔮 Fazit

Ein Smart Mirror mit Raspberry Pi kostet unter 300€ und ist ein echter Eye-Catcher. Plus: Alle Daten bleiben lokal — kein Tracking, keine Abhängigkeit von Internet. Perfekt für Tech-Enthusiasten!
