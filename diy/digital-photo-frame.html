WiFi Digitaler Bilderrahmen mit E-Ink

Ein energiesparender Bilderrahmen, der automatisch neue Bilder von Unsplash lädt — mit E-Ink-Display für papierähnliche Darstellung.

🖼️

Stell dir einen Bilderrahmen vor, der nie den Strom stecken muss (wenn er aus ist), der täglich neue Bilder zeigt, und der aussieht wie echtes Papier. Das ist ein E-Ink-Bilderrahmen.

💡 Warum E-Ink?

Ein normales LCD-Display verbraucht auch im Standby Strom. E-Ink braucht Strom NUR beim Bildwechsel — danach bleibt das Bild ohne Strom erhalten. Das macht ihn perfekt für einen Bilderrahmen, der 24/7 läuft.

Schritt 1: Hardware zusammenbauen

1

E-Ink HAT auf den Pi stecken

Stecke das E-Ink HAT direkt auf die GPIO-Pins des Raspberry Pi Zero 2 W. Kein Löten nötig — es sitzt einfach auf.

2

Software installieren

Flash Raspberry Pi OS Lite auf die SD-Karte und aktiviere SSH:

# Nach dem ersten Boot: sudo apt update && sudo apt upgrade -y sudo apt install python3-pip git -y # E-Ink Bibliothek installieren: pip3 install waveshare-epd

Schritt 2: Python-Skript erstellen

3

Auto-Update Skript

Erstelle ein Python-Skript, das täglich ein neues Bild von Unsplash lädt und auf dem E-Ink-Display anzeigt:

#!/usr/bin/env python3 import requests from PIL import Image import waveshare_epd # Neues Bild von Unsplash laden response = requests.get( "https://source.unsplash.com/random/800x480/?nature" ) img = Image.open(BytesIO(response.content)) img = img.resize((640, 384)) # Auf E-Ink anzeigen epd = waveshare_epd.EPD() epd.init() epd.display(epd.getbuffer(img)) print("Bild aktualisiert!")

Schritt 3: Automatisierung

Das Skript soll täglich um 8:00 Uhr laufen. Dafür nutzen wir cron:

# Crontab öffnen: crontab -e # Füge diese Zeile hinzu: 0 8 * * * /home/pi/epk-frame/update.py
