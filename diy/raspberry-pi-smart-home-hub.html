Smarter Zuhause-Hub mit Raspberry Pi 5

Baue deinen eigenen Smart-Home-Hub mit Home Assistant — komplett lokal, ohne Cloud. Deine Daten bleiben bei dir.

🏠

Willkommen zu diesem Tutorial! Wir bauen einen kompletten Smart-Home-Hub mit dem Raspberry Pi 5 und Home Assistant. Das Ergebnis: Du hast volle Kontrolle über alle deine smarten Geräte — ohne Cloud, ohne Abos, ohne Datenschutzbedenken.

💡 Warum lokal?

Cloud-basierte Smart-Home-Systeme wie Alexa oder Google Home senden deine Daten an Server in den USA. Mit Home Assistant bleibt ALLES in deinem Netzwerk. Kein Tracking, keine Abhängigkeit von Internet.

Was du brauchst

📦 Materialliste

  • 🛒 Raspberry Pi 5 (4GB oder 8GB) ~70€
  • 🛒 MicroSD-Karte (32GB, Class 10) ~8€
  • 🛒 Pi 5 Gehäuse mit Kühlung ~15€
  • USB-C Netzteil (27W) ~12€
  • 🛒 Zigbee/Dongle (ConBee II oder Sonoff Zigbee 3.0) ~25€
  • Ethernet-Kabel ~3€

Gesamtkosten: ~133€

Schritt 1: Home Assistant OS installieren

1

Home Assistant Image herunterladen

Lade das Home Assistant OS Image von home-assistant.io herunter.

Dann mit Raspberry Pi Imager auf die MicroSD-Karte schreiben:

# Raspberry Pi Imager öffnen # 1. OS wählen → "Home Assistant" → "Home Assistant OS" # 2. Storage wählen → deine MicroSD-Karte # 3. "Write" klicken # 4. Warten bis fertig
2

Erster Start

Stecke die SD-Karte in den Pi 5, verbinde Ethernet und schließe das Netzteil an. Nach ca. 2-3 Minuten ist Home Assistant bereit.

Rufe http://homeassistant.local:8123 in deinem Browser auf.

Schritt 2: Zigbee-Dongle einrichten

3

Dongle anschließen und aktivieren

Stecke den Zigbee-Dongle in einen USB-Port des Pi 5. In Home Assistant:

Einstellungen → Add-ons → Add-on Store → "Zigbee" suchen → "Zigbee MQTT Add-on" installieren → Starten → "Configure" → "Enable Zigbee" klicken

Dein Pi 5 ist jetzt ein Zigbee-Coordinator — die Basis für alle Zigbee-Geräte.

Schritt 3: Geräte hinzufügen

Jetzt kannst du deine ersten Geräte koppeln. So geht's bei den gängigsten Typen:

  • Zigbee-Sensoren (Temperatur, Bewegung, Türkontakt): Gerät 60 Sekunden lang in Pairing-Modus versetzen → Home Assistant findet es automatisch
  • Zigbee-Leuchtmittel (Philips Hue, IKEA Trådfri): Einschalten, 5x ein/aus schalten → Pairing-Modus → automatisch erkannt
  • Z-Wave-Geräte: Ähnlich wie Zigbee, benötigt aber einen Z-Wave-Dongle
  • Matter/Geräte: Direkt über Home Assistant einrichtbar

Schritt 4: Automatisierungen erstellen

Das Schöne an Home Assistant: Du kannst komplexe Automatisierungen erstellen, die lokal in Millisekunden reagieren — ohne Cloud-Latenz.

🎯 Beispiel-Automatisierungen

• Wenn Bewegung im Flur zwischen 23:00-06:00 → Licht auf 10%
• Wenn Temperatur unter 18°C → Heizung an
• Wenn Tür aufgeht und niemand zu Hause → Alarm
• Wenn Sonne scheint und Fenster offen → Benachrichtigung

Schritt 5: Fernzugriff (optional)

Falls du von unterwegs auf dein Smart Home zugreifen willst, gibt es zwei sichere Optionen:

  • Home Assistant Cloud (kostenlos für Basis, 5€/Monat für Premium)
  • tailscale (kostenlos, P2P-Verschlüsselung, keine Cloud)

🔮 Fazit

Ein eigener Smart-Home-Hub mit dem Raspberry Pi 5 kostet unter 150€ und gibt dir die volle Kontrolle über dein Zuhause. Keine monatlichen Gebühren, keine Cloud, keine Datenschutzprobleme. Der Einstieg dauert ca. 1-2 Stunden — aber der Spaß daran ewig.
