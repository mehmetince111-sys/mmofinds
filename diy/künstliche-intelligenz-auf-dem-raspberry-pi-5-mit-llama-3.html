KI auf dem Raspberry Pi 5: Llama 3 lokal betreiben

Betriebe Llama 3, Googles Gemma und andere KI-Modelle direkt auf deinem Raspberry Pi 5 — vollständig lokal und kostenlos

🎯 Warum KI auf dem Raspberry Pi?

🔒

100% Datenschutz

Alle Daten bleiben auf deinem Raspberry Pi. Kein Cloud, kein Tracking, keine Datensammlung. Deine Gespräche sind wirklich privat.

💰

Kostenlos

Keine Abo-Gebühren, keine API-Kosten. Einmal Raspberry Pi kaufen, dann für immer kostenlose KI.

⚡

Offline nutzbar

Kein Internet? Kein Problem. Die KI läuft komplett lokal auf deinem Raspberry Pi.

🎓

Lernen & Experimentieren

Perfektes Projekt, um KI-Technologie von innen zu verstehen. Open-Source-Modelle, die du anpassen kannst.

🛒 Benötigte Hardware

Raspberry Pi 5 (4GB oder 8GB)

Der Raspberry Pi 5 ist der leistungsstärkste Pi aller Zeiten. 4GB reichen für kleinere Modelle, 8GB sind empfehlenswert für Llama 3 8B.

Speicher: MicroSD-Karte (32GB+)

Eine schnelle MicroSD-Karte ist wichtig für die Performance. Class 10 oder UHS-I empfohlen.

📋 Schritt-für-Schritt-Anleitung

Schritt 1: Raspberry Pi OS installieren

Lade Raspberry Pi Imager herunter und installiere Raspberry Pi OS Lite (64-bit) auf deiner MicroSD-Karte:

  1. Installiere Raspberry Pi Imager von raspberrypi.com
  2. Wähle Raspberry Pi 5 als Gerät
  3. Wähle Raspberry Pi OS Lite (64-bit)
  4. Gehe auf Settings (Ctrl+Shift+X) und aktiviere SSH
  5. Setze Benutzername (pi) und Passwort
  6. Schreibe das Image auf die MicroSD-Karte

Schritt 2: Raspberry Pi einrichten

Stecke die MicroSD-Karte in den Pi, verbinde Keyboard, Monitor und Netzteil. Nach dem Booten:

# System aktualisieren
sudo apt update && sudo apt upgrade -y

# Abhängigkeiten installieren
sudo apt install -y python3 python3-pip git cmake build-essential

# RAM für Swap erhöhen (wichtig für KI-Modelle)
sudo sed -i 's/^CONF_SWAPSIZE=.*/CONF_SWAPSIZE=2048/' /etc/dphys-swapfile
sudo systemctl restart dphys-swapfile

Schritt 3: Ollama installieren

Ollama ist der einfachste Weg, lokale KI-Modelle zu betreiben:

# Ollama installieren
curl -fsSL https://ollama.com/install.sh | sh

# Ollama-Dienst starten
systemctl --user start ollama

# Llama 3 8B herunterladen (ca. 5GB)
ollama pull llama3.2:3b

# Für 8GB Pi: Llama 3 8B (größer, aber intelligenter)
ollama pull llama3.2:8b

Schritt 4: KI testen

Jetzt kannst du mit deiner lokalen KI chatten:

# Im Terminal chatten
ollama run llama3.2:3b

# Oder über Web-Interface
# Installiere Open WebUI für eine schöne UI
docker run -d -p 3000:8080 --add-host=host.docker.internal:host-gateway -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:main

Schritt 5: Verfügbare Modelle

Neben Llama 3 gibt es viele weitere Modelle, die auf dem Raspberry Pi 5 laufen:

Modell Größe Empfehlung
llama3.2:3b ~2 GB ✅ 4GB Pi — Schnell
llama3.2:8b ~5 GB ✅ 8GB Pi — Intelligent
gemma2:2b ~1.5 GB ✅ 4GB Pi — Sehr schnell
qwen2.5:7b ~5 GB ✅ 8GB Pi — Multilingual

🏆 Unser Urteil

Der Raspberry Pi 5 ist das perfekte Einsteiger-Gerät für lokale KI. Mit Llama 3 und anderen Modellen hast du eine voll funktionsfähige KI direkt zu Hause — ohne Cloud, ohne Abo, mit vollem Datenschutz. Für 4GB-Pi-Nutzer empfehlen wir llama3.2:3b oder gemma2:2b. Für 8GB-Nutzer ist llama3.2:8b die beste Wahl.
