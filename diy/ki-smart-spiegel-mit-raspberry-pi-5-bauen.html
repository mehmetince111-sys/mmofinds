KI-Smart Mirror mit Raspberry Pi 5 bauen

Der ultimative Smart Mirror: Wetter, Kalender, News, KI-gestützte Gesichtserkennung und Sprachsteuerung — alles auf einem 24-Zoll-Spiegeldisplay. Komplett lokal und kostenlos!

🔧 Schritt 1: Raspberry Pi einrichten

Lade das neueste Raspberry Pi OS Lite (64-bit) von der offiziellen Website herunter und spiele es mit Raspberry Pi Imager auf eine microSD-Karte (mindestens 32GB, Klasse 10).

💡 Pro-Tipp: Aktiviere SSH und WiFi direkt im Raspberry Pi Imager unter "Settings" — so kannst du den Pi später komplett ohne Monitor konfigurieren.

Stecke die Karte ein, verbinde den Pi mit Monitor und Tastatur, und führe die folgenden Befehle aus:

$ sudo apt update && sudo apt full-upgrade -y
$ sudo apt install -y git npm nodejs
$ sudo apt install -y chromium-browser
$ sudo apt install -y python3-pip
$ pip3 install --user pillow pyyaml
$ git clone https://github.com/MichMich/MagicMirror
$ cd MagicMirror && npm install

🪞 Schritt 2: Spiegel-Display vorbereiten

Reinige die Vorderseite des Monitors gründlich und trage die halbverspiegelfolie gleichmäßig auf. Verwende eine Sprühflasche mit Wasser und Spülmittel als Gleitmittel, um Blasen zu vermeiden.

Schneide überschüssige Folie mit einem scharfen Cutterserver entlang der Monitorränder ab. Der Effekt: Von vorne siehst du den Spiegel, von hinten erkennst du den Monitor — perfekt für den Smart Mirror!

🤖 Schritt 4: KI-Funktionen hinzufügen

Hier wird's spannend! Wir installieren Face Recognition für personalisierte Begrüßung und Whisper für lokale Sprachsteuerung:

$ pip3 install --user face-recognition
$ pip3 install --user openai-whisper
$ pip3 install --user pyaudio
$ sudo apt install -y portaudio19-dev
# Whisper Modell herunterladen (~2GB)
$ python3 -m whisper.download --model base

Erstelle ein Python-Skript mirror-ai.py für die KI-Steuerung:

#!/usr/bin/env python3
"""KI-Smart Mirror Steuerung"""
import whisper
import face_recognition
import subprocess
import json
import cv2

# Whisper für Sprachsteuerung
model = whisper.load_model("base")

# Gesichtserkennung initialisieren
known_faces = {}  # {name: encoding}

def recognize_face(frame):
    """Erkennt bekannte Gesichter im Bild"""
    rgb_frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
    face_locations = face_recognition.face_locations(rgb_frame)
    face_encodings = face_recognition.face_encodings(rgb_frame, face_locations)
    
    for face_encoding in face_encodings:
        for name, known_encoding in known_faces.items():
            matches = face_recognition.compare_faces([known_encoding], face_encoding)
            if matches[0]:
                return name
    return "Unbekannt"

def voice_command(text):
    """Verarbeitet Sprachbefehle"""
    text = text.lower()
    if "wetter" in text:
        return get_weather()
    elif "kalender" in text:
        return get_calendar()
    elif "licht an" in text:
        return control_smart_light("on")
    elif "licht aus" in text:
        return control_smart_light("off")
    return "Befehl nicht erkannt"

if __name__ == "__main__":
    print("KI-Smart Mirror bereit!")
    print("Sprich: 'Wetter', 'Kalender', 'Licht an'")

🏠 Schritt 5: Gehäuse bauen

Für das Gehäuse kannst du entweder 3D-gedruckte Teile verwenden (kostenlos auf Thingiverse oder Printables) oder ein einfaches Holzgehäuse aus Sperrholz bauen. Wichtig:

  • Sorge für ausreichend Belüftung — der Raspberry Pi wird warm
  • Verwende ein 24V Netzteil für den Monitor und einen USB-C Lader für den Pi
  • Integriere einen PIR-Bewegungsmelder (HC-SR501), damit der Spiegel sich einschaltet, wenn jemand in der Nähe ist
  • Verstecke alle Kabel im Gehäuse für ein sauberes Erscheinungsbild

⚠️ Wichtige Hinweise

  • 🔌 Verwende immer ein qualitatives Netzteil für den Raspberry Pi 5 — mindestens 27W USB-PD
  • 🌡️ Der Pi 5 sollte ein Kühler mit Lüfter haben, besonders im geschlossenen Gehäuse
  • 🔒 Sichere den Spiegel an der Wand — er kann bis zu 10kg wiegen

🎉 Fertig! Dein KI-Smart Mirror ist bereit

Herzlichen Glückwunsch! Du hast deinen eigenen KI-gestützten Smart Mirror gebaut. Er zeigt Wetter, Kalender, Nachrichten und lässt sich per Sprachbefehl steuern — alles komplett lokal ohne Cloud!

Du kannst den Mirror weiter erweitern mit:

  • 📊 Home Assistant Dashboard — steuere alle Smart Home Geräte direkt am Spiegel
  • 📸 Foto-Rahmen — zeige persönliche Fotos vom Spiegel an
  • 🎵 Musiksteuerung — Spotify oder YouTube Music per Sprachbefehl
  • 🤖 LLM-Chat — frage den Spiegel nach Informationen (mit lokal laufendem LLM)
