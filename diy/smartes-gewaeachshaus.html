Smartes Gewächshaus mit Raspberry Pi

Baue ein vollautomatisches Gewächshaus mit Temperaturüberwachung, automatischer Bewässerung und Smartphone-Benachrichtigungen — alles gesteuert vom Raspberry Pi.

🌱

Willkommen zu diesem Projekt! Wir bauen ein smartes Gewächshaus, das selbstständig Temperatur, Luftfeuchtigkeit und Bodenfeuchte misst, automatisch bewässert und dich per E-Mail oder Telegram benachrichtigt, wenn etwas nicht stimmt. Perfekt für Kräuter, Tomaten oder kleine Pflanzen — ohne dass du Zuhause sein musst.

🌿 Was das smarte Gewächshaus kann

• Echtzeit-Temperatur und Luftfeuchtigkeit messen
• Bodenfeuchte überwachen und automatisch bewässern
• Web-Dashboard im lokalen Netzwerk
• Benachrichtigungen per E-Mail oder Telegram
• Daten-Logging mit Historie (CSV-Export)
• Solarbetriebs-Option für den Garten

Schritt 1: Raspberry Pi vorbereiten

1

Operating System installieren

Lade Raspberry Pi OS Lite (64-bit, ohne Desktop — spart Ressourcen) herunter und schreibe es mit dem Raspberry Pi Imager auf die MicroSD-Karte.

# Raspberry Pi Imager öffnen # 1. OS → Raspberry Pi OS (other) → Raspberry Pi OS Lite (64-bit) # 2. Storage → deine MicroSD-Karte # 3. "Write" klicken # 4. Nach dem Write: Pi Imager Settings öffnen # → SSH aktivieren → Username: pi, Passwort: dein Passwort # → WLAN konfigurieren (SSID + Passwort)

Stecke die SD-Karte in den Pi 4, verbinde Ethernet (optional) und schließe das Netzteil an. Der Pi bootet in ca. 30 Sekunden.

Schritt 2: Sensoren anschließen

2

GPIO-Verdrahtung

Verbinde die Sensoren wie folgt mit dem Raspberry Pi 4 (GPIO-Pin-Belegung):

DHT22 Sensor: Pin 1 (VCC) → GPIO 2 (3.3V) Pin 2 (Data) → GPIO 4 (GPIO-24) Pin 3 (NC) → frei Pin 4 (GND) → GPIO 6 (GND) Soil Moisture Sensor: VCC → GPIO 2 (3.3V) GND → GPIO 6 (GND) AO (Analog Out) → GPIO 40 (ADC über MCP3008, siehe unten) DO → frei (nicht verwendet) Relay Modul (2-Kanal): VCC → GPIO 2 (3.3V) GND → GPIO 6 (GND) IN1 (Pumpe) → GPIO 17 (GPIO-17) IN2 (optional) → GPIO 27 (GPIO-27) MCP3008 (ADC für Analog-Sensor): VDD → GPIO 2 (3.3V) VREF → GPIO 2 (3.3V) AGND → GPIO 6 (GND) DGND → GPIO 6 (GND) CS → GPIO 8 (GPIO-10 / SPI CS0) CLK → GPIO 11 (GPIO-11 / SPI CLK) DOUT → GPIO 9 (GPIO-9 / SPI MISO) DIN → GPIO 10 (GPIO-8 / SPI MOSI) CH0 → AO vom Soil Sensor

⚡ Hinweis zum MCP3008

Der Raspberry Pi hat keine analogen Eingänge. Der MCP3008 wandelt analoge Signale (vom Soil Sensor) in digitale Werte um, die der Pi lesen kann. Er wird über SPI angesprochen — das ist beim Pi 4 standardmäßig aktiviert.

Schritt 3: Software einrichten

3

Python-Environment und Bibliotheken

Verbinde dich per SSH mit dem Pi und installiere die benötigten Bibliotheken:

# SSH verbinden: ssh pi@homeassistant.local (oder Pi-IP) # Python und pip aktualisieren sudo apt update && sudo apt upgrade -y sudo apt install python3 python3-pip -y # Bibliotheken installieren pip3 install RPi.GPIO adafruit-circuitpython-dht adafruit-blinka pip3 install flask requests # SPI aktivieren (falls nicht schon geschehen) sudo raspi-config # → Interface Options → SPI → Enable
4

Das Hauptprogramm schreiben

Erstelle die Datei /home/pi/greenhouse.py mit folgendem Inhalt:

#!/usr/bin/env python3 """ Smartes Gewächshaus — Hauptprogramm Misst Temperatur, Luftfeuchtigkeit, Bodenfeuchte und steuert die automatische Bewässerung. """ import time import RPi.GPIO as GPIO import adafruit_dht import board import json import datetime import os # --- Konfiguration --- DHT_PIN = board.D24 # GPIO 24 (DHT22 Data) PUMP_PIN = 17 # GPIO 17 (Relay IN1) TEMP_MIN = 10.0 # Min. Temperatur (°C) TEMP_MAX = 35.0 # Max. Temperatur (°C) MOISTURE_MIN = 300 # Min. Bodenfeuchte (ADC-Wert) MOISTURE_MAX = 800 # Max. Bodenfeuchte (ADC-Wert) LOG_INTERVAL = 300 # Logging alle 5 Minuten (Sekunden) PUMP_DURATION = 10 # Bewässerungsdauer (Sekunden) # --- GPIO Setup --- GPIO.setmode(GPIO.BCM) GPIO.setwarnings(False) GPIO.setup(PUMP_PIN, GPIO.OUT) GPIO.output(PUMP_PIN, GPIO.HIGH) # Relay deaktiv (inverted) # --- DHT22 Sensor --- dht = adafruit_dht.DHT22(DHT_PIN) # --- MCP3008 ADC (einfache Implementation) --- import spidev spi = spidev.SpiDev() spi.open(0, 0) def read_adc(channel): r = spi.xfer2([1, (8 + channel) << 4, 0]) ad = ((r[1] & 3) << 8) + r[2] return ad # --- Logging --- def log_data(temp, hum, moist): entry = { "timestamp": datetime.datetime.now().isoformat(), "temperature": temp, "humidity": hum, "moisture": moist } log_file = "/home/pi/greenhouse_data.csv" with open(log_file, "a") as f: if os.path.getsize(log_file) == 0: f.write("timestamp,temperature,humidity,moisture\n") f.write(f"{entry['timestamp']},{temp},{hum},{moist}\n") # --- Bewässerung --- def water_plants(): moisture = read_adc(0) if moisture < MOISTURE_MIN: print(f"⚠️ Boden zu trocken ({moisture}). Bewässere für {PUMP_DURATION}s.") GPIO.output(PUMP_PIN, GPIO.LOW) # Pumpe AN time.sleep(PUMP_DURATION) GPIO.output(PUMP_PIN, GPIO.HIGH) # Pumpe AUS print("✅ Bewässerung abgeschlossen.") # --- Temperatur-Warnung --- def check_temperature(temp): if temp < TEMP_MIN: print(f"❄️ Kälte-Warnung: {temp}°C — zu kalt für Pflanzen!") # Hier könnte E-Mail/Telegram gesendet werden elif temp > TEMP_MAX: print(f"🔥 Hitze-Warnung: {temp}°C — zu heiß!") # --- Hauptloop --- print("🌱 Smartes Gewächshaus gestartet!") print(f" Temp: {TEMP_MIN}°C - {TEMP_MAX}°C") print(f" Feuchte: {MOISTURE_MIN} - {MOISTURE_MAX}") last_log = time.time() while True: try: temp = dht.temperature hum = dht.humidity moist = read_adc(0) if temp is not None and hum is not None: print(f"📊 Temp: {temp}°C | Luftfeuchte: {hum}% | Boden: {moist}") check_temperature(temp) water_plants() now = time.time() if now - last_log >= LOG_INTERVAL: log_data(temp, hum, moist) last_log = now except RuntimeError as e: print(f"⚠️ Sensor-Fehler: {e}") time.sleep(30) # 30 Sekunden zwischen Messungen

Schritt 4: Automatischer Start

5

Programm als Service einrichten

Das Programm soll beim Booten automatisch starten und bei einem Crash neu starten:

# Script ausführbar machen chmod +x /home/pi/greenhouse.py # systemd Service erstellen sudo nano /etc/systemd/system/greenhouse.service

Füge folgenden Service-Inhalt ein:

[Unit] Description=Smartes Gewächshaus After=network.target [Service] Type=simple User=pi WorkingDirectory=/home/pi ExecStart=/usr/bin/python3 /home/pi/greenhouse.py Restart=always RestartSec=10 [Install] WantedBy=multi-user.target
# Service aktivieren und starten sudo systemctl daemon-reload sudo systemctl enable greenhouse sudo systemctl start greenhouse # Status prüfen sudo systemctl status greenhouse # Logs lesen journalctl -u greenhouse -f

Schritt 5: Web-Dashboard (optional)

6

Live-Daten im Browser ansehen

Erstelle eine einfache Flask-Webseite, die die letzten Messwerte anzeigt:

#!/usr/bin/env python3 """ Greenhouse Dashboard — Flask Web-Server Zeigt Live-Temperaturen und historische Daten an. """ from flask import Flask, jsonify import csv import datetime app = Flask(__name__) @app.route("/") def dashboard(): return """ 🌱 Smartes Gewächshaus

🌱 Smartes Gewächshaus Dashboard

--°C
Temperatur
--%
Luftfeuchtigkeit
--
Bodenfeuchte
""" @app.route("/api/data") def api_data(): log_file = "/home/pi/greenhouse_data.csv" try: with open(log_file, "r") as f: reader = csv.DictReader(f) last_row = None for row in reader: last_row = row if last_row: return jsonify({ "temperature": float(last_row["temperature"]), "humidity": float(last_row["humidity"]), "moisture": int(last_row["moisture"]), "timestamp": last_row["timestamp"] }) except: pass return jsonify({"temperature": 0, "humidity": 0, "moisture": 0}) if __name__ == "__main__": app.run(host="0.0.0.0", port=5000)

Schritt 7: Solarbetrieb (optional)

8

Das Gewächshaus autark machen

Mit einem kleinen Solarpanel und einer Powerbank kannst du das Gewächshaus komplett autark betreiben — ideal für abgelegene Gartenstandorte:

  • Solarpanel 10W 5V — erzeugt genug Strom für den Pi und Sensoren
  • TP4056 Charger Board — lädt eine 18650 Li-Ion Zelle sicher
  • Powerbank oder 18650 Zelle — Puffer für Nacht und bewölktes Wetter
  • DC-DC Buck Converter 5V — stabilisiert die Spannung für den Pi

☀️ Solar-Benefit

Ein 10W Solarpanel erzeugt bei 6 Sonnenstunden ca. 60Wh. Der Raspberry Pi 4 verbraucht ca. 3-5W — das reicht für einen ganzen Tag, auch mit Puffer für die Nacht. Im Sommer funktioniert das Gewächshaus so komplett ohne Stromanschluss.

Tipps & Troubleshooting

🔧 Häufige Probleme

• DHT22 liest 0°C oder 100%: Prüfe die Verdrahtung. Der Data-Pin muss korrekt auf GPIO 24 sein. Pull-up-Widerstand (10kΩ) zwischen VCC und Data hilft.
• Pumpe läuft nicht: Relay-Modul kann inverted (LOW = AN) oder normal (HIGH = AN) sein. Teste beide Polungen.
• Soil Sensor falsche Werte: Capacitive Sensoren müssen kalibriert werden. Luft-Wert = MAX, Wasser-Wert = MIN. Deine Werte dazwischen.
• SSH verbindet nicht: Prüfe mit sudo nmap -sn 192.168.1.0/24 auf dem Router die Pi-IP.
• SPI funktioniert nicht: sudo raspi-config → Interface Options → SPI → Enable. Dann Neustart.

🌿 Pflanzen-Tipps

• Ideal für: Kräuter (Basilikum, Minze), Cherry-Tomaten, Salat, Paprika
• Temperatur-Ziel: 18-25°C tagsüber, nicht unter 10°C nachts
• Bewässerung: Lieber öfter kurz als einmal lang — Staunässe vermeiden!
• Belüftung: Ein kleines Computer-Lüfter im Gewächshaus verhindert Überhitzung

Dein smartes Gewächshaus kostet unter 150€ (mit Solar ca. 165€) und gibt dir die volle Kontrolle über die Wachstumsbedingungen deiner Pflanzen. Es misst automatisch, bewässert wenn nötig und warnt dich, wenn etwas nicht stimmt — sogar wenn du bei der Arbeit bist.
